# About This Website

**CyberSafe** is a beginner-friendly cybersecurity guide made to help people understand:

- How the internet works  
- What dangers exist online  
- How to protect themselves  
- What habits make them safer  

This website is written in simple language so **anyone can understand cybersecurity**, even without technical background.

Created using **GitHub Pages** + **Chirpy Theme**.
